# Efficy Joshna – Academic Portfolio

A modern, responsive personal portfolio for an aspiring chemistry research student. Built with HTML, CSS, and JavaScript (no frameworks).

## Quick start

1. Open `index.html` in a browser, or run a local server (e.g. `npx serve .`) for best behavior.
2. Replace placeholders with your details (see below).

## Customization

- **Photo**: Replace the hero image by adding your photo (e.g. `photo.jpg`) in this folder and in `index.html` change the hero `<img src="...">` to `src="photo.jpg"`.
- **Resume**: Add your PDF as `resume.pdf` in this folder. The “Download Resume” button already links to it.
- **Contact**: In `index.html`, update the Contact section:
  - `your.email@example.com` → your email
  - `+91 98765 43210` → your phone number

## Features

- Soft academic palette (cream, light brown, muted blue, grey)
- Serif headings (Cormorant Garamond), sans-serif body (Source Sans 3)
- Hero fade-in on load, section reveal on scroll, hover effects on cards and buttons
- Education timeline with animated dots, competency cards, experience grid, certification badges
- Languages with progress bars, hobbies with icons, contact form and floating background shapes
- Fully responsive layout

## Files

- `index.html` – structure and content
- `styles.css` – layout, colors, animations
- `script.js` – scroll reveal, form submit, smooth scroll
